## 智账 Sage v1.0.0

首个正式发布版本。

### ✨ 新增

- **关于应用**页面：应用简介、开发者信息与版本号
- Sage 品牌标识与各平台启动图标
- **统计** Tab：时间范围筛选、分组列表、排序与汇总
- 规则驱动的 **分析** 引擎：时段对比、分类变化、波动诊断、预算风险等
- **理财**记录类型：与收支结余分离，支持利率、到期日与 App 内提醒
- 理财管理页：本金/净存入统计、到期列表、年度趋势图
- 银行账单 PDF/CSV 导入与审核，原始行敏感字段脱敏展示
- 多账本、分类管理、预算、数据备份（Excel/PDF）

### 🔧 变更

- 各平台应用显示名称统一为 **Sage** / **智账**
- 底部导航「统计」「分析」命名调整
- 旧「理财」分类自动迁移为 `wealth` 类型

### 🐞 修复

- 时间范围面板圆角样式
- 分类管理窄屏文字溢出
- 账单导入跳过行标题显示问题

### 说明

- **Android**：首次安装若提示未知来源，请在系统设置中允许安装。
